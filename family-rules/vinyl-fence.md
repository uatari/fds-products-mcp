# Vinyl Fence Family Rules

- Default to `6'H x 8'W` white privacy unless the customer clearly specifies otherwise.
- Treat plain "vinyl fence" as panels by default.
- Gates match fence height; do not ask gate height separately.
- Keep posts, caps, and panels in the same collection/color family.
- Use lookup results as the source of truth for designs, colors, and prices.
- If a customer wants comparisons, omit the compared dimension and use returned facets/options.
- If multiple listed items have the same price, collapse the list.

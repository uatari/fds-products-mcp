# Composite Board Family Rules

- Default `unit_name=Board`.
- Derive `collection` from exact catalog colors when possible.
- If customer says only `Enhance`, clarify `Basics` vs `Naturals`.
- Do not invent color names; use exact catalog colors only.
- Edge compatibility:
  - `Square` -> 16 or 20 ft
  - `Grooved` -> 12, 16, or 20 ft
  - `12 ft` -> `Grooved` only
- If collection/color compatibility is invalid, treat it as non-existent.
- Use lookup results as authoritative; do not adjust invalid combinations.
